entreniebla




'entreniebla' (2014) is a short run, self-published book that contains diverse explorations of algorithmic visual poetry (codepoetry), screenshots of accidents that were processes in movement and the code of each poem, it activates the code as a space for poetic expression, finding a space for subjectivity and experience through words.


* It was possible to create it only thanks to the help and support of Angela Sánchez de Vera and Ana Mayoral at Universidad Nacional Autónoma de México.
